engine-assets
==============================================================================

*Rails Engines with assets*

engine-assets is a Rails Engine which enables other Rails Engines to provide
assets:

  * javascript
  * css
  * images (TODO)


Requirements
------------------------------------------------------------------------------

  * Rails  
    supported versions:

    * 3.0.0 (requires Ruby 1.8.7)
